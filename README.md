#SinGraphic

有一段时间特别想在Terminal下做动态效果，但查资料又不知道关键字，花了很长时间才找到shell中的tput命令。

## 操作说明

sinGraphic.sh是shell调用sinGraphic程序，其中包含gcc编译sinGraphic.c，所以直接运行这个文件，就能看到程序效果。
